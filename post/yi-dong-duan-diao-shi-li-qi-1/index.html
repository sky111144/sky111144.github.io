Android 拾遗：Android Debug Bridge

2019-11-11

Android Debug Bridge，简称 adb，是安卓前端开发的利器，能让开发者在 PC 上高效的进行开发调试工作。

调试准备工作

  1. USB 连接电脑和手机。
  2. 开启手机开发者模式。各个厂商定制的安卓手机系统，开发者模式的有不同开启方式，搜索引擎搜索对应手机的教程即可。
  3. 检测是否连接成功。
# 查看连接设备列表
adb devices

录制手机屏幕

# 注意：录制视频的默认录制时间是 180 秒
# 注意：录制视频的最长录制时间是 180 秒

# 开始录屏，以 mp4 格式保存在 /sdcard
adb shell screenrecord /sdcard/demo.mp4

# 停止录屏：Control + C

文件管理

复制手机文件到电脑
# 命令格式
adb pull <手机设备文件路径> <电脑设备文件目录>

# 将手机文件 /sdcard/demo.mp4 复制到电脑文件夹 /tmp 下
adb pull /sdcard/demo.mp4 /tmp
