前端动画方案（三）：视频

2019-11-13

自从浏览器支持 HTML5 标准后，video 在动画领域提供了新的解决方案，扩展了开发者在解决业务问题的工具箱。

视频动画内容简介

video 是浏览器原生支持的媒体标签，支持 mp4、m3u8、ogg 三种视频格式，如今已经成为 Web 视频领域的游戏选择方案。
  • 标准格式
    • MP4：MP4 文件使用 H264 视频编解码器和 AAC 音频编解码器
    • WebM：WebM 文件使用 VP8 视频编解码器和 Vorbis 音频编解码器
    • Ogg：Ogg 文件使用 Theora 视频编解码器和 Vorbis 音频编解码器
  • 兼容性
浏览器 MP4 WebM Ogg
IE YES NO NO
Chrome YES YES YES
Firefox YES YES YES
Safari YES NO NO
Opera YES YES YES

视频动画场景

  • 长动画，内容复杂，需要以视频的形式播放的动画。

视频动画使用示例

  • video.js
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>video 使用示例</title>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link href="https://unpkg.com/video.js/dist/video-js.min.css" rel="stylesheet">
    <script src="https://unpkg.com/video.js/dist/video.min.js"></script>
  </head>
  <body>
    <video id="my-player" class="video-js" controls preload="auto" data-setup='{}'>
        <source src="//www.runoob.com/try/demo_source/mov_bbb.mp4" type="video/mp4"></source>
        <source src="//www.runoob.com/try/demo_source/mov_bbb.ogg" type="video/ogg"></source>
    </video>
  </body>
</html>
