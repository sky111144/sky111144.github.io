前端动画方案：SVG

2019-11-14

SVG 是可伸缩矢量图形 (Scalable Vector Graphics)，浏览器兼容性好，图形绘制功能好用，但有一定的学习成本。

SVG 内容简介

SVG 技术特点

  • SVG 使用 XML 格式定义图形，SVG 文件是纯粹的 XML。
  • SVG 图像在放大或改变尺寸的情况下其图形质量不会有所损失。
  • SVG 于 2003 年 1 月 14 日成为 W3C 推荐标准，各个浏览器兼容性好。

SVG 使用场景

  • 矢量动画，能用几何知识规则描述的动画。

SVG 使用方式

  1. embed 标签
  2. object 标签
  3. iframe 标签
  4. a 标签
  5. img 标签
  6. HTML 内联
  7. 背景图片

SVG 绘图类型

  1. 矩形
  2. 圆形
  3. 椭圆
  4. 直线
  5. 多边形
  6. 曲线
  7. 路径
  8. 文本
  9. 图片

SVG 特效类型

  1. 填充（Stroke）
  2. 滤镜（Filter）
  3. 模糊（Blur）
  4. 阴影（Shadow）
  5. 渐变（Gradient）
